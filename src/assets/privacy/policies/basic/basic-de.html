Datenschutzerklärung für z-control QR Code Generator App

Gültigkeitsdatum: 30. Juni 2025

Version: 1.0

🔒 Datenschutz & Datensicherheit

Diese Datenschutzerklärung erklärt, wie die z-control QR Code Generator App Ihre Daten sammelt, verwendet und schützt.

📱 Datensammlung

  • Keine persönlichen Daten: Wir sammeln keine persönlichen Daten, außer Sie stellen sie explizit bereit
  • E-Mail-Adressen: Werden nur lokal auf Ihrem Gerät für die bequeme QR-Code-Zustellung gespeichert
  • QR-Code-Inhalt: Eingegebener Text wird lokal verarbeitet und nicht an unsere Server übertragen
  • Kein Tracking: Wir verwenden keine Analysen, Cookies oder Tracking-Technologien

💾 Lokale Datenspeicherung

  • Gerätespeicher: Alle Daten (E-Mail-Adressen, Einstellungen) werden lokal auf Ihrem Gerät gespeichert
  • Automatische Bereinigung: Generierte QR-Code-Dateien werden nach 3 Stunden automatisch gelöscht
  • Benutzerkontrolle: Sie können alle gespeicherten Daten jederzeit über die App-Einstellungen löschen

🌐 Web-App vs. Mobile App

  • Web-App: Verwendet Browser-Local-Storage, keine Server-Kommunikation
  • Mobile App: Verwendet geräteeigenen IndexedDB-Speicher
  • Keine Cloud-Synchronisation: Daten werden niemals zwischen Geräten synchronisiert

📧 E-Mail-Funktionalität

  • Lokale Verarbeitung: QR-Codes werden lokal auf Ihrem Gerät generiert
  • E-Mail-Integration: Verwendet Ihre Geräte-E-Mail-App - wir handhaben keine E-Mail-Übertragung
  • Keine E-Mail-Überwachung: Wir können Ihre E-Mails nicht sehen oder darauf zugreifen

🔓 Open-Source-Transparenz

Diese App ist Open Source. Sie können unseren Code auf GitHub überprüfen, um unsere Datenschutzpraktiken zu verifizieren.

👶 Datenschutz für Kinder

Unsere App ist für Benutzer aller Altersgruppen geeignet. Wir sammeln wissentlich keine persönlichen Informationen von Kindern unter 13 Jahren ohne elterliche Zustimmung.

🔄 Änderungen an dieser Richtlinie

Wir können diese Datenschutzerklärung von Zeit zu Zeit aktualisieren. Wir werden Benutzer über wesentliche Änderungen über die App oder unsere Website informieren.

📞 Kontaktinformationen

Bei Datenschutzfragen oder -bedenken kontaktieren Sie: z-control Support
