z-control Multi Language Translator App
What the app does

Enter text up to {{ parameters?.maxInputLength }} characters and translate it into up to {{ parameters?.maxTargetLanguages }} languages using Google Translate, then have it read aloud. Both the base language (English or German) and target languages can be freely selected in the settings.

Ideal for travelers, language learners, or anyone needing quick translations into multiple languages. Use it to improve your language skills or communicate more effectively while traveling abroad.

Highlights:

  • Simultaneous translation into multiple languages at once, allowing you to practice and use several languages in parallel for maximum flexibility.
  • Text-to-speech reads translations aloud, making comprehension easier and supporting pronunciation learning.
  • Smart completion: Google Translate can automatically complete incomplete sentences in target languages.
  • Privacy-focused: Translations are processed on Google servers; this app does not store your text.

This web version is hosted on Google Firebase.

 What are the limitations and monthly quotas?
Limitations:

Shared free monthly quota: {{ parameters?.maxTranslateCharsTotalPerMonth | number: "1.0-0" : "en" }} characters for all users. If this quota is exceeded, costs will be incurred for z-control.

To manage this shared quota, each user is limited to {{ parameters?.maxTargetLanguages }} target languages and {{ parameters?.maxTranslateCharsUserPerMonth | number: "1.0-0" : "en" }} characters per month, ensuring fair access for all users.

Since the Google Translate API is a paid service, quotas are necessary. The app will notify you well in advance when approaching the limit and will prevent further translations once the monthly quota is exhausted.

Text-to-speech is completely free and unlimited. You can listen to translations as many times as you want without affecting your monthly quota.

How do I check my monthly translation quota?

On the translation page

Your current translation quota is displayed at the bottom of the translation page in an expandable Monthly Quota section.

When either your personal quota or the shared quota for all users is reached, translations are simulated. All other features (such as text-to-speech) remain fully functional. Quotas automatically reset on the 1st of each month.

Important: More target languages and longer texts consume your quota faster. Choose target languages and text length wisely to maximize your monthly usage.

On the Statistics page in Settings

View detailed, anonymized statistics showing the overall monthly quota usage, device information, and target language preferences across all users. All data is aggregated to protect your privacy.

Differences between Web and Mobile App
Mobile Version Advantages:

Use voice input directly through your device's keyboard for hands-free text entry.

Customize speech output speed and pitch in the settings for optimal listening.

Web Version Advantages:

Comfortable typing with a full keyboard in your browser.

Voice input is available on some operating systems—Windows 11 users can activate it with Windows key + H.

The web version supports text-to-speech for all languages, while the mobile version has limited language availability.

IMPORTANT

When accessing the web version on your phone, ensure it displays in mobile mode (not desktop mode) for optimal experience. Mobile mode provides a native app-like interface and enables voice input through your keyboard.
