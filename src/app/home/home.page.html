z-control QR Code Generator App
What the app does

The app creates a QR code from the text you enter and sends it by email. Input is plain text (no formatting) up to {{ maxInputLength }} characters. You can enter any text or paste a URL.

The QR code is attached to the email as both an image (PNG) and a PDF, and the original text is included in the message so you can easily identify each QR code.

Print options: Choose the QR code size in the print settings. The system can fill an A4 page or print a specific number of QR codes per page.

Save email addresses: Store frequently used addresses so you don’t need to re-enter them each time.

This web version is hosted on Google Firebase.

 What are QR Codes and how can you use them?

QR codes (Quick Response Codes) are two-dimensional barcodes that store information in a square pattern. They can be read with a camera or QR code scanner and allow quick access to information, websites, or actions.

What can z-control QR Code Generator app do?

This app allows you to generate QR codes from any plain text you enter. You can use it to:

  • Share simple messages or notes: Encode any text and let others scan it to read your message.
  • Share links: Paste a website URL as plain text, and the QR code will open the link when scanned.
  • Label items: Print QR codes with custom text to label documents, objects, or locations.
  • Transfer information: Quickly move text between devices by scanning the QR code.

Note: Unlike some other QR code apps, this app does not support special QR code formats for contacts, emails, Wi-Fi, or calendar events. All QR codes generated by this app contain only the plain text you enter.

How does it work?
  1. Enter your text: Type or paste any text (up to {{ maxInputLength }} characters).
  2. Generate the QR code: Tap the "Generate" button to create a QR code from your text.
  3. Send or print:
    • You can email the QR code as an image or PDF, or print it for use on labels, cards, or documents.
    • You can also share the QR code via messaging apps or social media.
    • You can define the size and number of the printed QR code in the print settings.
  4. Scan to retrieve: Anyone can scan the QR code with a standard QR code reader to view the plain text you entered.
Why use plain text QR codes?
  • Simplicity: No special formatting or technical knowledge required.
  • Privacy: The app does not store or process your data externally.
  • Flexibility: Use QR codes for any purpose where sharing plain text is helpful.

In summary: This app is designed for users who want a simple, reliable way to generate and share QR codes containing plain text—no more, no less.

Differences between Web and Mobile App and Structure of the app

You can use z-control QR Code Generator as Web App in your browser or as mobile app installed from the Google Play Store. There is only one source code for both apps.

App Structure & Navigation

The app uses two main tabs for navigation:

  • QR Code: For creating and sending QR codes.
  • Settings: For managing QR code size for PDFs, email addresses, language settings, and viewing app information.

Navigation

On mobile devices in portrait mode: Use the tabs at the bottom of the screen to switch between QR Code and Settings.

On tablets, desktops, or in landscape mode: Use the navigation button in the top right of the header to switch between QR Code and Settings.

The web app offers advantages for creating longer QR code texts, but it has limitations compared to the mobile app.

Differences between Web and Mobile App
Mobile App

On Android devices the system picker opens so you can choose your preferred mail app. This choice can be reset in the Android settings.

The generated QR codes are saved in the "Documents" folder and automatically attached to the email. When the app starts, QR code files older than 3 hours are deleted automatically.

The email body includes the full QR code text and more detailed printing instructions.

The app is available on the Google Play Store and can be used offline. Emails are sent automatically once an internet connection is available again.

Web-specific information is omitted from this help page — consult the web app for platform-specific details and instructions.

Web App

The browser or operating system opens the registered email client. On desktop systems this is either the system's default mail client or a webmail interface configured in the browser.

For privacy and security reasons browsers cannot attach files automatically to emails. Therefore you must manually add the QR code files from the Downloads folder as attachments. Browsers also cannot delete files automatically; cleanup must be done manually.

The email body includes references to both QR code files (PDF and PNG), the QR code text, and shortened printing instructions. Because sending is done via mailto:, the mail body is length limited; typically only the first ~400 characters (of a maximum of 1000) of the QR code text and a reduced printing instruction are included.

The Web App is platform-independent and requires no installation. It can be used directly in the browser but requires an internet connection.
