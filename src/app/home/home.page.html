z-control QR Code App
What can the app do?

A QR code is created from the entered text and sent by email. The text cannot be formatted and can be up to {{ maxInputLength }} characters long. Text or a link to a web page can be entered.

The QR code is attached to the email as an image and PDF, and the entered text is included in the email. This way you can immediately see which text is contained in the QR code, in case you create multiple QR codes.

To avoid having to enter the email address every time you send an email, you can save email addresses.

This web version is hosted on Google Firebase.

 Run it as Web App

You can use or test the app here without installing it on your phone.

Limitations when using the app as a web app

    Since browsers are not allowed to delete data, you must manually delete the saved QR codes in the download directory if you wish.

    Since webmail clients are not allowed to automatically add attachments, the saved QR codes must be manually added to the generated email.

    In the mobile app, this is all done automatically.
